Ubuntu上安装anaconda

Created2023-07-01|Updated2023-07-01|技术分享
|Post View:

  • 官网下载 https://www.anaconda.com/安装包

  • 运行.sh 文件

    bash Anaconda3-2021.11-Linux-x86_64.sh
/home/lm/anaconda3 # 安装路径

  • 配置环境

    sudo vim ~/.bashrc
export PATH=/home/lm/anaconda3/bin:$PATH
source ~/anaconda3/bin/activate #修改终端的默认 python 为 anaconda

  • 保存环境配置

    source ~/.bashrc

  • 查看版本

    conda --version

  • 虚拟环境

    conda create -n env_name python=3.8

  • 激活环境

    conda activate env_name

  • 查看环境

    conda info --envs
或
conda info -e
或
conda env list

  • 复制环境

    conda create --name new_env_name --clone copied_env_name

  • 删除环境

    conda remove --name env_name --all
